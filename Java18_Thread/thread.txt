Program : (mac 의 경우 app) 응용소프트웨어. (.exe or .app) : 실행할수있는 파일을 의미. // 메모장이라는 어플.얘를 프로그램이라함.
Process : 프로그램이 실행되고 있는 객체 (메모리에 올라간 객체)  // 메모장 어플을 키면 4개를 키면 4개가 켜짐. 이 4개를 프로세스라함.
Thread : 하나의 process에서 여러개의 메소드 단위로 실행되는 모듈. = 기능. ( 프로세스 안에서 돌고있는 기능)


java에서 메소드 단위로 프로그램을 실행하는 방법 2가지
1.runnable을 implements 한다.(단순 thread화.)
2.Thread 를 extends한다. (여러개의 Thread 기능을 사용해야 할 때)
